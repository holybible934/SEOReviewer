【樂天1.25"HOT" DOG年】國內刷卡消費 登錄享1.25%回饋金

  • 如何參加
  • 活動詳情

貼心小提醒

  1. 本活動期間內，正、附卡持有人於國內刷卡一般消費合併計算，且正卡人於本公司會員服務頁面完成登錄，即可獲得刷卡金回饋資格。(國內刷卡一般交易認定詳見活動注意事項)
  2. 活動期間新增之一般刷卡金額(排除餐廳消費)加總後1.25%刷卡金回饋。刷卡金以回饋至正卡人帳戶為準，回饋金額無上限。刷卡金將直接折抵信用卡消費帳款，不得折換現金、兌換其他商品或轉讓予第三人。
  3. 活動消費交易以本公司帳單列示消費日及金額為準，且需於2018/4/20前完成入帳。
  4. 刷卡金回饋將於2018年6月4日統一回饋至正卡人樂天信用卡帳戶後於帳單揭露。
  5. 提醒您 欲享有餐廳消費回饋，請詳見《【樂天饕友饗宴】全台餐廳3%回饋火燙延燒》活動注意事項。

活動詳情說明

活動名稱
【樂天1.25"HOT" DOG年】國內刷卡消費 登錄享1.25%回饋金
活動期間
2018年3月1日~2018年3月31日
活動詳細辦法
  1. 本活動期間內，正、附卡持有人於國內刷卡一般消費合併計算，且正卡人於本公司會員服務頁面登錄後，即可獲得刷卡金回饋資格。(國內刷卡一般交易認定詳見活動注意事項)
  2. 活動期間新增之一般刷卡金額(排除餐廳消費)加總後1.25%刷卡金回饋。刷卡金以回饋至正卡人帳戶為準，回饋金額無上限。
  3. 活動消費交易以本公司帳單列示消費日及金額為準，且需於2018/4/20前完成入帳
  4. 刷卡金回饋將於2018年6月4日統一回饋至正卡人樂天信用卡帳戶後於帳單揭露。
  5. 本活動刷卡金將直接折抵信用卡消費帳款，不得折換現金、兌換其他商品或轉讓予第三人。
  6. 提醒您 欲享有餐廳消費回饋，請詳見《【樂天饕友饗宴】全台餐廳3%回饋火燙延燒》活動注意事項。
參加活動範例說明

樂小姐於2018/03/14至樂天百貨消費NT3,000元，2018/03/21於樂天餐廳消費NT3,000元；附卡於2018/03/22網路購物消費NT5,000元，並於2018/03/26去日本旅遊購物刷卡NT5,000，因本活動排除餐廳消費且日本旅遊購物不屬於國內刷卡消費，符合活動資格之交易合計8000元，並於活動期間內在會員服務頁面登錄，且店家於2018/4/20內請款完畢，樂小姐可獲得8,000*1.25%=100元刷卡金。

刷卡金回饋時間
2018年6月4日
